Support

Need help with Dumb Keyboard? Start here.

Setup

  1. Open Settings > General > Keyboard > Keyboards.
  2. Tap Add New Keyboard....
  3. Select Dumb Keyboard.
  4. Tap Dumb Keyboard and enable Allow Full Access (required for haptics).

Common Issues

Keyboard not showing
Open any text field, tap the globe key, and select Dumb Keyboard.

No haptic feedback
Make sure Allow Full Access is enabled for Dumb Keyboard in iOS Settings.

Settings deep link doesn’t open
Some iOS versions block deep links. Open manually:
Settings > General > Keyboard > Keyboards

Data Collection
